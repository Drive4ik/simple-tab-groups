Configuring STG synchronization via Github Gist

What is this for?

This is to synchronize your tabs and Simple Tab Groups settings between two or more computers. Your data will be stored in a Github Gist file. This file will be private. Apart from you, no one will have access to it unless you yourself provide a Github Gist link to this data to another person. STG will only send this data to your Github account.

How do I set up synchronization?

  1. You need to create a Github account (if you don't have one), and confirm your email.
  2. You need to create a private token in your Github account. The token should only be stored by you and in the STG settings. It will be used to transfer your tabs to your Github account.

    The token must be given permission to write data to Gist!

    • Create fine-grained personal access token. See example.
    • or
    • Create classic token. See example
  3. Write the created token to the STG settings.
  4. Click the "Save settings" button
  5. Click the "Start synchronization" button
